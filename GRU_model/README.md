Dans ce dossier, vous trouverez différents fichiers :
- GRUIntentModel : Ce fichier est un vestige. Il applique seulement notre modèle GRU pour les intents.
- GRUIntentAndSlotModel : Ce fichier créé notre modèle avec ses composantes
- GRU_train_test : Ce fichier entraine notre modèle à partir d'un corpus contenant toutes les langues sur toutes les langues ; puis l'évalue
- GRU_train_test_zero_shot : Ce fichier entraine notre modèle sur un corpus contenant toutes les langues sauf le français sur toutes les langues sauf le français
- GRU_test_zero_shot : Ce fichier évalue le modèle entrainé sur tous sauf le français en le testant sur le français
