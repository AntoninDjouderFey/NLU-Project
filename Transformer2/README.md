Dans ce dossier, vous trouverez différents fichiers :
- TransformerModel : Ce fichier créer notre modèle avec ses composantes (comme les vecteurs d'embeddings)
- train_transformer : Ce fichier entraine notre modèle à partir d'un corpus contenant toutes les langues sur toutes les langues
- train_transformer_zero_shot : Ce fichier entraine notre modèle sur un corpus contenant toutes les langues sauf le français sur toutes les langues sauf le français
- test_transformer_zero_shot : Ce fichier évalue le modèle entrainé sur tous sauf le français en le testant sur le français
